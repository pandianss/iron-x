# Release v1.0.0: Foundation
**Date**: 2026-02-15
**Type**: MAJOR

## Summary
The initial release of the Iron-X Enterprise Platform, establishing the core monetization and governance infrastructure.

## Features
### 1. Monetization & Subscriptions
- **Tiers**: Free (Individual) vs. Pro (Team).
- **Limits**: Configurable limits on Actions and Goals per tier.
- **Admin**: Backend management API for subscription plans.

### 2. Compliance Mapping
- **Reports**: Automated SOC 2 and ISO 27001 readiness reports.
- **Evidence**: JSON/PDF export of audit trails.

### 3. Enterprise Features
- **SSO**: SAML 2.0 integration (Generic/Okta compatible).
- **Audit**: Comprehensive, immutable audit log export (CSV).
- **Policy Engine**: Hierarchical policy enforcement (Org > Dept > Team > User).

### 4. Sales Enablement
- **Collateral**: ROI Calculator, Pitch Deck, and Case Studies.
- **Website**: Public marketing site structure (`/`, `/solutions/*`).

## Implementation Details
- **Database**: Prisma schema extended with `Subscription`, `Tier`, `SSOConfig`, `Policy`, `AuditLog`.
- **Frontend**: React + Tailwind. New `MarketingLayout` and `DashboardLayout`.
- **Backend**: Express + TypeScript. Middleware for subscription enforcement.

## Migration Steps
1. Run `npx prisma migrate deploy` to apply schema changes.
2. Initialize default Plans via `db:seed` (if applicable) or manual admin API call.
